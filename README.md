# Authorized Host Auditor

A simple Python tool for checking hosts that you own or are authorized to test.

## Features

- DNS resolution
- TCP port checking
- HTTP/HTTPS status checking
- TLS certificate info
- Optional WebSocket upgrade test
- JSON report export

## Requirements

- Python 3
- pip
- requests
